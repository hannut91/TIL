## SCM

Software Configuration Management

## Git의 핵심

* Snapshot
* Local Database
* Integrity (SHA-1)
  - 무결성이 보장된다. 조금이라도 코드에 변경사항이 있으면 값도 변화한다.
* Only Add Data
  - 오직 추가만 가능하다.
* 3 States
  * Working Directory
  * Staging Area (index) -> Snapshot
  * Repository (.git)
